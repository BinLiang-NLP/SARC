# SARC
Evaluation code for the Self-Annotated Reddit Corpus.
  * https://arxiv.org/abs/1704.05579 (Manuscript)
  * http://nlp.cs.princeton.edu/SARC/2.0/ (Dataset)
  
Dependencies: NLTK, scikit-learn, text_embedding.
